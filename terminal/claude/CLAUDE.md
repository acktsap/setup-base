## Testing

- When writing unit tests, use the `write-london-unit-test` skill.

## Code Intent

- When writing or modifying code, use the `clarify-code-intent` skill.

## Java Documentation

- When writing or modifying Java files, evaluate and apply the Java documentation skills below as applicable.
- When writing or modifying Java package boundaries, use the `java-package-info-doc` skill.
- When writing or modifying Java public types, interfaces, SPI types, factories, or core domain objects, use the `java-class-javadoc` skill.
- When writing or modifying Java public APIs, protected APIs, interface methods, SPI methods, extension points, or complex methods, use the `java-method-javadoc` skill.
