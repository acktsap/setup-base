name: explain
description: Explain code with clear structure and examples. Use when asking how code works or what it does.
---

When explaining code:

1. Start with a one-line summary of what it does
2. Break down the main components or steps
3. Explain key logic and algorithms
4. Note any edge cases or gotchas
5. Provide usage example if applicable

Keep explanations concise and use bullet points.
